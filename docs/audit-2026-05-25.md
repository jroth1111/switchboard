## Residual risks (not fixed)

1. **Planner `deployment-filter`** — Not integrated into live request path; suspect/quarantine/RPM parity with admission deferred until wiring.
2. **Fixture worker** — `/responses` and `/messages` error scenarios incomplete vs chat completions (low impact for current smoke paths).
3. **`classifyProviderFailure`** — `400` + `"token"` still maps to `context_length_exceeded` globally (mitigated for ChatGPT-specific paths).
4. **`reapAllLeases`** — Only touches single control-plane DO stub (name vs behavior in tests).
5. **`scripts/failures-probe.ts`** — Untracked file left out of audit commits (remove or adopt separately).
6. **OAuth DO tests** — No round-trip encryption tests without vitest `ENCRYPTION_KEY` configuration.
